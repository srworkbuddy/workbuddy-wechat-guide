🏢 关于 WorkBuddy

WorkBuddy 是腾讯 CSIG 智慧零售自研的 AI 智能办公平台，已服务内部用户 2,000+，集成 20+ 专业技能包与 5 大主流大模型，覆盖智能问答、文档生成、数据分析、业务流程自动化等核心办公场景。

WorkBuddy 支持 PC 网页端、企业微信、个人微信 三端协同，实现跨端消息互通与会话连续，确保工作上下文在任意终端无缝衔接。

本文档提供企业微信与个人微信两种接入方式的标准化操作指引，帮助团队在 10 分钟内完成部署与绑定，快速启用 AI 办公能力。

3 选择接入方式并绑定

企微提供两种接入方式，推荐长连接模式：

推荐

长连接模式

配置最简单，只需填 Bot ID 和 Secret

  • ✅ 步骤最少
  • ✅ 无需回填 URL
  • ✅ 连接更稳定
备选

URL 回调模式

适用于需要 Webhook 回调的场景

  • 需填写 Token / AESKey
  • 需回填 Webhook URL
  • 配置步骤更多

🔗 长连接模式步骤

  1. 在企微「API 配置」区域选择「使用长连接」
  2. 复制 Bot ID
  3. 点击「点击获取」获取 Secret
  4. 打开 WorkBuddy → 右上角头像 → Claw 设置
  5. 在「集成 (BETA)」找到 「企微 AIBot 集成」 → 点击「配置」
  6. 选择 「WebSocket 长连接」，填入 Bot ID 和 Secret
  7. 点击「注册」完成绑定

⚠️ 长连接模式不需要填写 URL、Token 或 Encoding-AESKey。复制凭据时避免带入多余空格。

🔗 URL 回调模式步骤

  1. 在企微「API 配置」区域选择「使用 URL 回调」
  2. 点击 Token 和 Encoding-AESKey 右侧的「随机获取」并保存
  3. 打开 WorkBuddy → Claw 设置 → 企微 AIBot 集成 → 配置
  4. 切换到「使用 URL 回调」，填入 Token 和 Encoding-AESKey
  5. 点击「注册」，复制生成的 Webhook URL
  6. 回到企微创建页面，将 Webhook URL 粘贴到 URL 输入框
  7. 点击「保存」完成配置
4 联通测试
  1. 在企微通讯录「企业创建的」分组下找到刚创建的机器人
  2. 点击「发消息」，发送一条简单消息（如"你好"）
  3. WorkBuddy 收到消息并返回回复 = 对接成功 🎉

❓ 常见问题
🔍 机器人没有响应？
  1. 检查 WorkBuddy 是否正在运行
  2. 确认 Claw 服务已开启
  3. 核对企业微信与 WorkBuddy 中选择的是同一种接入方式
  4. 长连接：核对 Bot ID 和 Secret；回调：核对 Token 和 Encoding-AESKey
  5. 检查电脑网络连接是否正常
🔍 URL 验证失败？
  1. 确保 WorkBuddy 处于运行状态
  2. 检查 Claw 服务是否正常启动
  3. 重新复制 Webhook URL，确保没有遗漏或多余字符
  4. 确认 Token 和 Encoding-AESKey 与 WorkBuddy 中配置完全一致
🔍 长连接注册失败？
  1. 确认企业微信侧已选择「使用长连接」
  2. 重新复制 Bot ID 和 Secret，避免带入多余空格
  3. 若 Secret 已失效，在企微后台重新获取后再次注册

❓ 常见问题
🔍 页面一直显示「绑定中...」？
  1. 确认 WorkBuddy 正在运行且网络正常
  2. 关闭配置窗口后重新进入
  3. 如仍存在，重启 WorkBuddy 后重新绑定
🔍 扫码后没有显示「已绑定」？
  1. 确认使用的是目标微信账号
  2. 等待几秒让状态同步
  3. 若二维码失效，点击「重试」生成新二维码
🔍 微信发消息没有响应？
  1. 确认电脑 WorkBuddy 仍在运行
  2. 检查 Claw 服务未关闭
  3. 返回 Claw 设置验证绑定状态
  4. 必要时解绑后重新扫码绑定
🔍 如何更换绑定的微信账号？
在 Claw 设置的微信 ClawBot 集成卡片中点击「解绑」，然后重新扫码绑定新账号即可。
